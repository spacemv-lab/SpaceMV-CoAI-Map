# Map AI Web Application

基于 React + Vite + Cesium 构建的智能地图应用。

## 快速开始

启动应用：

```bash
pnpm nx serve map-ai
pnpm  nx serve map-ai-api
```

构建应用：

```bash
pnpm nx build map-ai
```

## 开发指南

详细的前端开发规范、组件使用说明及目录结构请参考：
[前端开发规范指南](../FRONTEND_GUIDE.md)

## 主要特性

- **UI 框架**: Shadcn/ui + TailwindCSS v4
- **地图**: Cesium (集成中)
- **架构**: Nx Monorepo Feature-Based Architecture
