# 点图层渲染策略设计与实现

## 1. 需求概述

### 1.1 核心需求

| 需求 | 描述 | 优先级 |
|------|------|--------|
| 二三维切换 | 支持 2D 符号（PointPrimitive）和 3D 广告牌（Billboard）切换 | P0 |
| 自定义符号 | 支持三角形、五角星等自定义形状 | P0 |
| 图片上传 | 支持用户上传自定义图标作为广告牌图片 | P0 |
| 文字标注 | 支持对点添加文字标签 | P0 |
| 精细化控制 | 控制单个点是否显示标签、是否可见等 | P1 |
| 性能优化 | 大数据量时使用高性能渲染方式 | P1 |

### 1.2 渲染策略总览

```
                        点图层渲染策略
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
   渲染模式选择          符号形状生成        文字标注控制
        │                    │                    │
   ┌────┴────┐          ┌────┴────┐         ┌────┴────┐
   │         │          │         │         │         │
Point    Billboard   Canvas   上传图片   全局标签  单点控制
Primitive  (广告牌)   生成               显示/隐藏  显示/隐藏
   │         │          │                    │
   │         │          │                    │
高性能    高表现力      2D 符号                精细化
(10w+ 点)   (自定义)    (三角形/星形)          控制
```

```typescript
  /** 创建/更新点集合 */

function createPointRenderer(mode: PointRenderMode): IPointRenderer {

      return new PointPrimitiveRenderer();
    case 'billboard':
      return new BillboardRenderer();
    case 'model':
      return new ModelRenderer(); // 预留
    default:
```

## 4. 详细实现方案

### 4.1 PointPrimitive 渲染器（高性能模式）

```typescript
// renderer/point-primitive-renderer.ts

import * as Cesium from 'cesium';
import { LayerState } from '../types/map-state';

export class PointPrimitiveRenderer implements IPointRenderer {
  private pointCollection: Cesium.PointPrimitiveCollection | null = null;
  private labelCollection: Cesium.LabelCollection | null = null;
  private featureIdMap: Map<string, Cesium.PointPrimitive> = new Map();
  private labelIdMap: Map<string, Cesium.Label> = new Map();

  update(layer: LayerState, viewer: Cesium.Viewer): void {
    // 1. 如果渲染模式变化，需要重建
    if (!this.pointCollection) {
      this.pointCollection = viewer.scene.primitives.add(
        new Cesium.PointPrimitiveCollection()
      );
    }

    // 2. 处理标签集合
    const hasLabels = !!layer.style.label?.text;
    if (hasLabels && !this.labelCollection) {
      this.labelCollection = viewer.scene.primitives.add(
        new Cesium.LabelCollection()
      );
    } else if (!hasLabels && this.labelCollection) {
      viewer.scene.primitives.remove(this.labelCollection);
      this.labelCollection = null;
      this.labelIdMap.clear();
    }

    // 3. 清空现有集合（简单实现，可优化为增量更新）
    this.pointCollection.removeAll();
    this.featureIdMap.clear();

    if (!layer.data?.features) return;

    // 4. 批量添加点
    const points = layer.data.features.map(feature => {
      const [lng, lat] = (feature.geometry as any).coordinates;
      const override = layer.featureOverrides?.[feature.id];

      const point = this.pointCollection!.add({
        position: Cesium.Cartesian3.fromDegrees(lng, lat),
        pixelSize: layer.style.pointSize || 10,
        color: Cesium.Color.fromCssColorString(
          override?.styleOverride?.color || layer.style.color || '#ffffff'
        ).withAlpha(
          override?.styleOverride?.opacity ?? layer.style.opacity ?? 1
        ),
        outlineColor: Cesium.Color.fromCssColorString(
          layer.style.pointOutlineColor || '#000000'
        ),
        outlineWidth: layer.style.pointOutlineWidth || 0,
        heightReference: Cesium.HeightReference.CLAMP_TO_GROUND,
        scaleByDistance: new Cesium.NearFarScalar(
          1.5e2, 1.0,    // 近处：150m 时缩放 1.0
          20e3, 0.5      // 远处：20km 时缩放 0.5
        ),
        show: override?.visible !== false,
      });

      this.featureIdMap.set(feature.id, point);

      // 添加标签
      if (this.labelCollection && layer.style.label?.text) {
        const showLabel = override?.showLabel ?? true;
        if (showLabel) {
          const label = this.addLabel(
            feature,
            layer,
            override?.labelOverride
          );
          this.labelIdMap.set(feature.id, label);
        }
      }

      return point;
    });

    return points;
  }

  private addLabel(
    feature: any,
    layer: LayerState,
    override?: Partial<LabelStyle>
  ): Cesium.Label {
    const labelStyle = layer.style.label!;
    const [lng, lat] = (feature.geometry as any).coordinates;

    return this.labelCollection!.add({
      position: Cesium.Cartesian3.fromDegrees(lng, lat),
      text: override?.text || labelStyle.text || '',
      font: override?.font || labelStyle.font || '14px sans-serif',
      fillColor: Cesium.Color.fromCssColorString(
        override?.fillColor || labelStyle.fillColor || '#ffffff'
      ),
      outlineColor: Cesium.Color.fromCssColorString(
        override?.outlineColor || labelStyle.outlineColor || '#000000'
      ),
      outlineWidth: override?.outlineWidth || labelStyle.outlineWidth || 0,
      style: Cesium.LabelStyle[
        override?.style || labelStyle.style || 'FILL'
      ],
      horizontalOrigin: Cesium.HorizontalOrigin[
        override?.horizontalOrigin || labelStyle.horizontalOrigin || 'CENTER'
      ],
      verticalOrigin: Cesium.VerticalOrigin[
        override?.verticalOrigin || labelStyle.verticalOrigin || 'CENTER'
      ],
      pixelOffset: new Cesium.Cartesian2(
        override?.pixelOffset?.[0] || labelStyle.pixelOffset?.[0] || 0,
        override?.pixelOffset?.[1] || labelStyle.pixelOffset?.[1] || 0
      ),
    });
  }

  setFeatureVisible(featureId: string, visible: boolean): void {
    const point = this.featureIdMap.get(featureId);
    if (point) {
      point.show = visible;
    }
  }

  setFeatureLabelVisible(featureId: string, visible: boolean): void {
    const label = this.labelIdMap.get(featureId);
    if (label) {
      label.show = visible;
    }
  }

  destroy(): void {
    this.featureIdMap.clear();
    this.labelIdMap.clear();
    // 注意：不销毁 collection，由调用方管理
  }

  getFeatureId(featureId: string): string {
    return featureId;
  }
}
```

### 4.2 Billboard 渲染器（高表现力模式）

```typescript
// renderer/billboard-renderer.ts

import * as Cesium from 'cesium';
import { LayerState } from '../types/map-state';
import { createSymbolCanvas } from '../utils/symbol-canvas';

export class BillboardRenderer implements IPointRenderer {
  private billboardCollection: Cesium.BillboardCollection | null = null;
  private labelCollection: Cesium.LabelCollection | null = null;
  private featureIdMap: Map<string, Cesium.Billboard> = new Map();
  private labelIdMap: Map<string, Cesium.Label> = new Map();
  private imageCache: Map<string, string | HTMLCanvasElement> = new Map();

  async update(layer: LayerState, viewer: Cesium.Viewer): Promise<void> {
    // 1. 确保 billboardCollection 存在
    if (!this.billboardCollection) {
      this.billboardCollection = viewer.entities.add().billboardCollection;
    }

    // 2. 处理标签
    const hasLabels = !!layer.style.label?.text;
    if (hasLabels && !this.labelCollection) {
      this.labelCollection = viewer.entities.add().labelCollection;
    } else if (!hasLabels && this.labelCollection) {
      this.labelCollection.removeAll();
      this.labelCollection = null;
      this.labelIdMap.clear();
    }

    // 3. 清空现有
    this.billboardCollection.removeAll();
    this.featureIdMap.clear();

    if (!layer.data?.features) return;

    // 4. 准备图片资源
    const imageSource = await this.prepareImageSource(layer);

    // 5. 批量添加广告牌
    for (const feature of layer.data.features) {
      const [lng, lat] = (feature.geometry as any).coordinates;
      const override = layer.featureOverrides?.[feature.id];

      const billboard = this.billboardCollection.add({
        position: Cesium.Cartesian3.fromDegrees(lng, lat),
        image: imageSource,
        scale: layer.style.pointSizeUnit === 'meters'
          ? undefined
          : (layer.style.pointSize || 1) / 32, // 假设原图 32x32
        width: layer.style.pointSizeUnit === 'meters'
          ? layer.style.pointSize
          : undefined,
        height: layer.style.pointSizeUnit === 'meters'
          ? layer.style.pointSize
          : undefined,
        sizeInMeters: layer.style.pointSizeUnit === 'meters',
        color: Cesium.Color.fromCssColorString(
          override?.styleOverride?.color || layer.style.color || '#ffffff'
        ).withAlpha(
          override?.styleOverride?.opacity ?? layer.style.opacity ?? 1
        ),
        rotation: (layer.style.pointRotation || 0) * Math.PI / 180,
        horizontalOrigin: Cesium.HorizontalOrigin[
          layer.style.pointHorizontalOrigin || 'CENTER'
        ],
        verticalOrigin: Cesium.VerticalOrigin[
          layer.style.pointVerticalOrigin || 'CENTER'
        ],
        heightReference: Cesium.HeightReference.CLAMP_TO_GROUND,
        show: override?.visible !== false,
      });

      this.featureIdMap.set(feature.id, billboard);

      // 添加标签
      if (this.labelCollection && layer.style.label?.text) {
        const showLabel = override?.showLabel ?? true;
        if (showLabel) {
          const label = this.addLabel(feature, layer, override?.labelOverride);
          this.labelIdMap.set(feature.id, label);
        }
      }
    }
  }

  /**
   * 准备图片资源
   * 优先级：自定义上传 > Canvas 生成符号 > 默认图片
   */
  private async prepareImageSource(layer: LayerState): Promise<string | HTMLCanvasElement> {
    // 1. 自定义上传的图片
    if (layer.style.pointImageUri) {
      if (this.imageCache.has(layer.style.pointImageUri)) {
        return this.imageCache.get(layer.style.pointImageUri)!;
      }

      // 如果是 URL，直接使用
      if (layer.style.pointImageUri.startsWith('http')) {
        this.imageCache.set(layer.style.pointImageUri, layer.style.pointImageUri);
        return layer.style.pointImageUri;
      }

      // 如果是 dataURL，也直接使用
      if (layer.style.pointImageUri.startsWith('data:')) {
        this.imageCache.set(layer.style.pointImageUri, layer.style.pointImageUri);
        return layer.style.pointImageUri;
      }
    }

    // 2. Canvas 生成符号
    const symbol = layer.style.pointSymbol || 'circle';
    const color = layer.style.color || '#ffffff';
    const size = 64; // Canvas 尺寸

    const cacheKey = `symbol-${symbol}-${color}-${size}`;
    if (this.imageCache.has(cacheKey)) {
      return this.imageCache.get(cacheKey)!;
    }

    const canvas = createSymbolCanvas(symbol, color, size);
    this.imageCache.set(cacheKey, canvas);
    return canvas;
  }

  private addLabel(
    feature: any,
    layer: LayerState,
    override?: Partial<LabelStyle>
  ): Cesium.Label {
    // 实现与 PointPrimitiveRenderer 相同
    // ...
  }

  setFeatureVisible(featureId: string, visible: boolean): void {
    const billboard = this.featureIdMap.get(featureId);
    if (billboard) {
      billboard.show = visible;
    }
  }

  setFeatureLabelVisible(featureId: string, visible: boolean): void {
    const label = this.labelIdMap.get(featureId);
    if (label) {
      label.show = visible;
    }
  }

  destroy(): void {
    this.featureIdMap.clear();
    this.labelIdMap.clear();
    this.imageCache.clear();
  }

  getFeatureId(featureId: string): string {
    return featureId;
  }
}
```

### 4.3 Canvas 符号生成工具

```typescript
// utils/symbol-canvas.ts

/**
 * 根据符号类型生成 Canvas
 * @param symbol 符号类型
 * @param color 颜色（CSS 颜色字符串）
 * @param size Canvas 尺寸（像素）
 * @returns HTMLCanvasElement
 */
export function createSymbolCanvas(
  symbol: string,
  color: string,
  size: number = 64
): HTMLCanvasElement {
  const canvas = document.createElement('canvas');
  canvas.width = size;
  canvas.height = size;
  const ctx = canvas.getContext('2d')!;

  ctx.fillStyle = color;
  ctx.beginPath();

  const center = size / 2;

  switch (symbol) {
    case 'circle':
      ctx.arc(center, center, size / 2 - 2, 0, Math.PI * 2);
      break;

    case 'square':
      ctx.rect(2, 2, size - 4, size - 4);
      break;

    case 'triangle':
      ctx.moveTo(center, 2);
      ctx.lineTo(size - 2, size - 2);
      ctx.lineTo(2, size - 2);
      ctx.closePath();
      break;

    case 'diamond':
      ctx.moveTo(center, 2);
      ctx.lineTo(size - 2, center);
      ctx.lineTo(center, size - 2);
      ctx.lineTo(2, center);
      ctx.closePath();
      break;

    case 'star':
      drawStar(ctx, center, center, 5, size / 2 - 2, size / 4);
      break;

    case 'cross':
      const w = size / 5;
      ctx.rect(center - w / 2, 2, w, size - 4);
      ctx.rect(2, center - w / 2, size - 4, w);
      break;

    default:
      ctx.arc(center, center, size / 2 - 2, 0, Math.PI * 2);
  }

  ctx.fill();

  // 添加白色描边（增强对比度）
  ctx.strokeStyle = 'rgba(255, 255, 255, 0.5)';
  ctx.lineWidth = 2;
  ctx.stroke();

  return canvas;
}

/**
 * 绘制五角星
 */
function drawStar(
  ctx: CanvasRenderingContext2D,
  cx: number,
  cy: number,
  spikes: number,
  outerRadius: number,
  innerRadius: number
) {
  let rot = Math.PI / 2 * 3;
  let x = cx;
  let y = cy;
  const step = Math.PI / spikes;

  ctx.beginPath();
  ctx.moveTo(cx, cy - outerRadius);

  for (let i = 0; i < spikes; i++) {
    x = cx + Math.cos(rot) * outerRadius;
    y = cy + Math.sin(rot) * outerRadius;
    ctx.lineTo(x, y);
    rot += step;

    x = cx + Math.cos(rot) * innerRadius;
    y = cy + Math.sin(rot) * innerRadius;
    ctx.lineTo(x, y);
    rot += step;
  }

  ctx.lineTo(cx, cy - outerRadius);
  ctx.closePath();
}

/**
 * 将 Canvas 转换为 DataURL（用于缓存或上传）
 */
export function canvasToDataURL(canvas: HTMLCanvasElement, format = 'image/png'): string {
  return canvas.toDataURL(format);
}
```

### 5.1 要素级别覆盖配置

```typescript
// store/use-map-store.ts

// 新增 action
interface MapStoreActions {
  // ... 现有 actions

  // 要素级别控制
  setFeatureOverride: (
    layerId: string,
    featureId: string,
    override: Partial<FeatureOverride>
  ) => void;

  clearFeatureOverride: (
    layerId: string,
    featureId: string
  ) => void;

    layerId: string,
    overrides: Record<string, Partial<FeatureOverride>>
  ) => void;
}

// 实现
export const useMapStore = create<MapStoreState>()(
  immer((set, get) => ({
    // ... 现有实现

    setFeatureOverride: (layerId, featureId, override) =>
      set((state) => {
        const layer = state.layers.find((l) => l.id === layerId);
        if (layer) {
          if (!layer.featureOverrides) {
            layer.featureOverrides = {};
          }
          if (!layer.featureOverrides[featureId]) {
            layer.featureOverrides[featureId] = {};
          }
          layer.featureOverrides[featureId] = {
            ...layer.featureOverrides[featureId],
            ...override,
          };
        }
      }),

    clearFeatureOverride: (layerId, featureId) =>
      set((state) => {
        const layer = state.layers.find((l) => l.id === layerId);
        if (layer?.featureOverrides) {
          delete layer.featureOverrides[featureId];
        }
      }),

    batchSetFeatureOverrides: (layerId, overrides) =>
      set((state) => {
        const layer = state.layers.find((l) => l.id === layerId);
        if (layer) {
          layer.featureOverrides = {
            ...layer.featureOverrides,
            ...overrides,
          };
        }
      }),
  }))
);
```

## 8. 总结

### 8.1 技术方案对比

| 特性 | PointPrimitive | Billboard |
|------|----------------|-----------|
| 性能 | ⭐⭐⭐⭐⭐ (10w+ 点) | ⭐⭐⭐ (1w 左右) |
| 表现力 | ⭐⭐ (仅圆形) | ⭐⭐⭐⭐⭐ (任意图片) |
| 符号形状 | 仅圆形 | Canvas 生成任意形状 |
| 图片支持 | ❌ | ✅ |
| 文字标注 | ✅ | ✅ |
| 旋转支持 | ❌ | ✅ |
| 尺寸单位 | 像素 | 像素/米 |
| 3D 效果 | 2D 圆形 | 3D 广告牌 |

### 8.2 推荐使用场景

- **PointPrimitive**：大规模散点数据（如人口分布、传感器网络）
- **Billboard**：POI 标注、设施定位、自定义图标需求

### 8.3 关键特性

1. **二三维切换**：通过 `pointRenderMode` 配置无缝切换
2. **自定义符号**：Canvas 动态生成三角形、五角星等
3. **图片上传**：支持上传自定义图标，自动缓存
4. **文字标注**：全局配置 + 单点覆盖
5. **精细化控制**：每个点独立控制可见性、标签显示
