# 组件 Props 与事件

- MapViewer
  - props：socketUrl、className、style、enableTimeline、enableDrawing、enableCurtain
  - 事件：onViewerReady(viewer)、onLayersChange(layers)、onDrawEnd(geojson)、onTimeChange(clock)

# 协议（tool / ui_action / Ack）

- tool：{ id, type: 'geo' | 'imagery' | 'tileset' | 'ui_action', data, schema, conversationId }
- ui_action：
  - toggleLayer：{ target: { name|layerId }, params: { visible } }
  - setTime：{ params: { timeISO, speed?, playing? } }
  - drawStart/drawEnd：{ params: { geometryType, resultGeoJSON? } }
  - setSplit：{ params: { position(0..1), leftLayerId, rightLayerId } }
- Ack：uiActionAck({ actionId, status, message? })
