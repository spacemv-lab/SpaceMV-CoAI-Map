# 功能与UI设计

本文档描述了 GIS 地图核心功能模块（Map Core）的功能需求、UI 设计和交互流程。

## 1. 总体目标

实现一个轻量级的 WebGIS 地图核心组件，支持底图加载、矢量数据图层管理、图层样式配置以及与后端 GIS 数据服务的全链路打通。

## 2. 功能模块

### 2.1 地图容器 (Map Container)

- **功能**：加载 3D 地球（Cesium）或 2D 地图。
- **交互**：支持缩放、平移、旋转、倾斜等基本地图操作。
- **底图**：默认加载天地图或 OSM 底图。

### 2.2 图层管理器 (Layer Manager)

- **位置**：地图左上角悬浮面板。
- **功能**：
  - 展示当前已添加的图层列表。
  - **图层排序**：支持通过拖拽句柄（Drag Handle）调整图层叠放顺序。
  - 控制图层的显示/隐藏（Checkbox）。
  - 图层操作菜单（右键或更多按钮）：
    - **打开属性表**：查看图层属性数据。
    - **缩放至图层**：定位到图层范围。
    - **样式设置**：打开样式配置面板。
    - **删除图层**：移除该图层。
  - **添加空白图层**：支持创建一个自定义名称的空白图层。
- **UI 设计**：
  - 顶部标题栏："图层" + "添加图层" 按钮（Primary Color）+ "新建空白图层" 按钮。
  - 列表区域：扁平列表，每行包含 拖拽句柄 + 图标（点/线/面）+ 图层名称 + 显隐开关 + 操作按钮。

### 2.3 添加图层弹窗 (Add Layer Modal)

- **触发**：点击图层管理器的 "添加图层" 按钮。
- **功能**：
  - **我的数据** Tab：展示用户上传的 GIS 数据集（复用 GIS Data Manager 的数据源）。
  - **瓦片服务** Tab：支持添加 XYZ、WMS、WMTS 等标准服务。
- **列表展示**：
  - 表格形式：序号、文件名、几何类型、记录数、来源、上传时间。
  - **批量添加**：支持多选数据，一次性添加多个图层。
- **交互**：选中数据后点击 "确定"，调用后端接口获取 GeoJSON 并加载到地图。

### 2.4 样式配置面板 (Style Panel)

- **触发**：在图层管理器中点击 "样式设置"。
- **位置**：地图右侧抽屉（Drawer）。
- **约束**：强制每个图层仅支持单一几何类型，简化样式配置。
- **功能**：根据图层几何类型提供不同的配置项。
  - **预设样式 (Presets)**：提供点、线、面的常用样式模板，一键应用。
  - **通用**：显示层级（MinZoom, MaxZoom）。
  - **点图层**：颜色、大小（半径）、透明度、图标（可选）。
  - **线图层**：颜色、宽度、透明度、线型（实线/虚线）。
  - **面图层**：填充颜色、填充透明度、边框颜色、边框宽度。
- **交互**：修改配置后实时反馈到地图上。

### 2.5 属性表 (Attribute Panel)

- **触发**：在图层管理器图层菜单中点击 "打开属性表"。
- **入口联动**：
  - 图层管理器每个图层项的更多菜单中提供 "属性表" 开关项。
  - 点击后若属性表未打开，则打开属性表并切换到当前图层；若已打开，则切换到当前图层并高亮对应图层项。
  - 属性表的打开状态与 "编辑图层（activeLayer）" 解耦，不能复用编辑态；允许“正在编辑 A 图层，同时查看 B 图层属性表”。
  - 当属性表当前展示的图层与图层管理器中的图层一致时，该图层项右侧增加一个弱提示状态（如 `Table` 图标高亮 / 标签色点），用于表达“当前属性表正在查看此图层”。
  - 若当前展示图层被删除，属性表自动切换到下一个可用图层；若没有可用图层，则自动关闭。
- **位置**：地图底部停靠面板（Bottom Dock），默认展开高度 320px，支持折叠到 40px 标题栏，支持拖拽调整高度（建议范围 280px 到 60vh）。
- **布局避让**：
  - 属性表优先占据地图底部全宽区域，但不覆盖左下角状态栏与图例的可操作区域。
  - 当属性表展开时，`BottomBar` 固定上浮到属性表标题栏上方左侧，保持 8px 到 12px 间距。
  - 图例默认停靠在属性表左上方；若图例展开高度与属性表发生垂直重叠，则图例自动上移，不与属性表内容区叠压。
  - 当未来时间轴启用时，底部空间优先级为：属性表 > 时间轴 > BottomBar；时间轴可吸附在属性表上边缘，避免多层覆盖。
  - 在窄屏或低高度窗口中，属性表切换为覆盖式底部抽屉，默认 70vh，并自动隐藏图例到折叠按钮态。
- **组成**：
  - 顶部标题栏：图标 + `"属性表 - 图层名"` + 要素数量 + 视图切换（`数据表` / `表字段`）+ 折叠/关闭按钮。
  - 工具栏：搜索、过滤、定位选中、批量删除、导出（预留）。
  - 主体区域：根据当前 Tab 展示 `数据表` 或 `表字段`。
  - 底部状态栏：当前记录数 / 选中数 / 字段数 / 编辑状态提示。
- **核心功能**：
  - 面向单个图层展示属性结构与要素数据。
  - 支持在属性表内切换图层，默认展示从图层管理器打开的目标图层。
  - 属性编辑以“单元格级编辑”为主，字段结构管理以“字段级配置”为主。
  - 所有改动先更新前端图层状态；服务端持久化能力若后续接入，沿用相同交互。
- **视图一：数据表（Records）**：
  - 一列对应一个属性字段，一行对应一个要素。
  - 第一列为选择列，用于多选要素；第二列固定为要素 ID / 系统编号。
  - 默认展示全部字段，字段过多时支持横向滚动、字段固定、字段显隐。
  - 支持关键字搜索（跨当前图层所有字段）和条件过滤（指定字段、操作符、值）。
  - 点击单元格进入编辑态；按字段类型提供对应编辑控件：
    - 字符串：文本输入框。
    - 数值：数字输入框。
    - 布尔：开关或下拉。
    - 日期：日期选择器（若后续支持）。
  - 支持行级操作：定位到要素、缩放到要素、删除要素。
  - 支持批量操作：删除选中要素。
  - 在地图上选中要素时，属性表自动滚动并高亮对应行；在属性表点选行时，地图同步高亮对应要素。
- **视图二：表字段（Fields）**：
  - 以字段定义表格展示当前图层的属性结构。
  - 支持新增字段、删除字段、修改字段配置。
  - 字段配置包含：
    - 名称
    - 别名
    - 类型
    - 长度
    - 非空
    - 索引
    - 备注
  - 字段名称要求在单图层内唯一；删除字段需要二次确认。
  - 字段类型变更属于高风险操作：
    - 若存在不兼容历史值，需要提示用户并阻止直接保存，除非用户执行数据清洗或确认转换策略。
    - 长度仅对字符串等有限长类型生效，无效时置灰。
  - 新增字段后，`数据表` 立即新增一列并为历史记录填入默认空值。
- **交互规则**：
  - 属性表关闭仅关闭面板，不影响图层可见性、编辑态、地图选择集。
  - 折叠后保留当前图层与当前 Tab 状态，便于再次展开继续操作。
  - 切换图层时保留 Tab 类型，但清空上一个图层的筛选条件和勾选状态。
  - 若图层没有任何要素：
    - `数据表` 显示空状态，并保留“新增字段”“新增要素（预留）”入口。
    - `表字段` 仍可正常管理字段结构。
  - 若图层没有任何属性字段：
    - `数据表` 提示先到 `表字段` 新增字段。
    - `表字段` 提供空状态引导。
- **与现有模块的关系**：
  - `LayerManager` 负责“打开哪个图层的属性表”。
  - `Attribute Panel` 负责“当前属性查看上下文（attributeLayerId）”。
  - `activeLayerId` 继续只表示绘制/编辑归属图层，不承担属性表状态。
  - `selection` 继续表示地图当前选中要素，属性表与其双向联动。

### 2.6 交互与编辑 (Interaction & Editing)

- **画笔操作**：
  - **位置**：地图容器的右侧，三个按钮按列排开（点、线、面）。
  - **归属权**：
    - 绘制的元素归属于当前 "激活"（编辑模式）的图层。
    - 若无激活图层，自动创建并激活 "我的标注" 图层。
- **要素编辑**：
  - **点要素**：支持拖拽改变位置。
  - **线/面要素**：支持编辑顶点（移动、插入、删除节点）。
- **高亮与弹窗**：
  - **高亮**：鼠标悬停在要素上时，要素高亮显示。
  - **弹窗 (Popup)**：
    - 点击要素时，在地图上显示属性弹窗。
    - 支持同时显示多个弹窗。
    - 弹窗内容包含要素的属性信息。

### 2.7 时间轴

- **位置**：地图容器的底部，水平方向。
- **功能**：展示当前地图上所有已添加的图层的时间范围。
- **UI 设计**：
  - 一个有颜色的长轴，一个轴对应一个图层，这个图层的一个时刻对应一个点，这个点下面标注了时间。
  - 鼠标悬浮到该轴上时，可以通过左键按住拖动显示的时间范围。
  - 点击一个时间点后会高亮。
  - 有多个图层就显示多个轴。
- **交互**：
  - 勾选卫星影像图层，识别到这个图层是多时相的，自动加载时间轴。
  - 时间轴上会有点，每个点对应一个时刻，这个点下面标注了时间。
  - 鼠标点击该点上时，会显示该时刻的影像。
  - 勾选多个有时间信息的图层就显示多个轴。

### 2.8 地图工具栏 (Map Toolbar)

- **位置**：地图容器的右上角或右下角（根据布局调整）。
- **功能按钮**：
  - **全屏**：切换地图全屏显示。
  - **截图保存**：保存当前地图视图为图片。
  - **底图切换**：切换不同的底图服务（如天地图、OSM、卫星图）。
  - **缩放控制**：放大/缩小按钮。
  - **图例**：显示图层图例。
  - **比例尺**：显示地图比例尺。
  - **缩放级别**：显示当前缩放等级。
  - **鼠标坐标**：实时显示鼠标所在位置的经纬度坐标。

## 3. 交互流程 (User Flow)

1.  **初始化**：用户进入地图页面，看到底图和空的图层管理器。
2.  **添加数据**：
    - 用户点击 "添加图层"。
    - 系统弹出 "添加图层" 对话框，选择 "我的数据" Tab。
    - 用户勾选 "成都市-贵阳"（LineString）和 "POI数据"（Point）等多个数据集，点击确定。
    - 系统批量请求后端接口获取 GeoJSON 并加载到地图。
    - 图层管理器列表中新增对应图层。
3.  **调整样式**：
    - 用户觉得默认线条颜色不明显。
    - 在图层列表中点击 "成都市-贵阳" 的 "样式设置"。
    - 右侧弹出样式面板。
    - 用户选择一个 "红色粗线" 的预设样式，或手动修改颜色为红色、宽度为 3px。
    - 地图上的线条实时更新。
4.  **管理图层**：
    - 用户拖拽图层列表项，调整图层叠放顺序。
    - 用户暂时不想看该图层，取消勾选显隐开关，图层隐藏。
5.  **查看、编辑图层属性**：
    - 鼠标悬停在地图要素上，要素高亮。
    - 点击地图上的要素，弹出属性弹窗（Popup），展示详细信息。
    - 若需查看完整属性表，在图层菜单中点击 "打开属性表"，底部弹出属性表抽屉。
    - 若当前图层处于 "激活" 状态，用户可直接拖拽点要素或编辑线/面要素的顶点。

## 4. UI 细节规范

- **配色**：遵循系统整体配色（shadcn/ui 默认风格）。
- **图标**：使用 `lucide-react` 图标库。
  - 图层：`Layers`
  - 添加：`Plus`
  - 设置：`Settings`
  - 删除：`Trash2`
  - 显隐：`Eye` / `EyeOff`
  - 点：`MapPin`
  - 线：`Activity`
  - 面：`Square` / `Polygon`
  - 拖拽：`GripVertical`
  - 全屏：`Maximize` / `Minimize`
  - 截图：`Camera`
  - 底图：`Map`
- **排版**：
  - 面板背景采用半透明磨砂效果（`bg-background/80 backdrop-blur`），提升地图通透感。
  - 字体大小适中（14px），保证信息密度。

## 5. 图层操作菜单功能详细设计

### 5.1 开启编辑 / 停止编辑

#### 5.1.1 功能说明

"开启编辑"功能允许用户对图层中的矢量要素进行几何编辑。同一时间只能有一个图层处于编辑状态。

#### 5.1.2 交互流程

1. **开启编辑**
   - 用户在图层操作菜单中点击 "开启编辑"。
   - 系统将该图层设置为编辑状态，其他图层的编辑状态自动取消。
   - 图层管理器中该图层项高亮显示（蓝色边框），名称前显示铅笔图标。
   - 地图上的该图层要素视觉反馈：
     - 图层整体高亮（使用更鲜明的颜色或加粗边框）。
     - 图层渲染顺序调整至最顶层，确保不被其他图层遮挡。
   - 鼠标指针在地图要素上时变为对应编辑形状。

2. **点图层编辑**
   - **默认状态**：所有点要素以高亮样式显示（如放大 1.2 倍，边框高亮）。
   - **悬停交互**：鼠标移动到点上时，指针变为 "拖动" 形状（grab），点要素进一步高亮。
   - **拖动交互**：
     - 按住鼠标左键拖动点要素。
     - 拖动过程中显示半透明的 "幽灵" 点跟随鼠标。
     - 释放鼠标后，点要素位置更新。
     - 支持撤销操作（Ctrl+Z）。

3. **线图层编辑**
   - **默认状态**：线条高亮显示（加粗、颜色鲜明）。
   - **节点显示**：
     - 点击线段后，显示所有现有节点（使用空心圆或实心圆标识）。
     - 节点可拖动，拖动后线段形状实时更新。
   - **插入节点**：
     - **方式一（节点间点击）**：在两个现有节点之间的线段上点击，在该位置创建新节点。
     - **方式二（吸附创建）**：鼠标靠近线段时，自动吸附到线上，点击创建新节点。
       - 吸附范围：距离线段 10 像素内触发。
       - 吸附位置显示预览点。
   - **删除节点**：
     - 右键点击节点，弹出菜单选择 "删除节点"。
     - 或选中节点后按 Delete 键删除。
     - 至少保留 2 个节点（构成一条线段的最小要求）。

4. **面图层编辑**
   - 交互方式与线图层一致。
   - 额外支持：
     - 点击面内部可拖动整个面（所有顶点同时移动）。
     - 首尾节点自动闭合显示。

5. **停止编辑**
   - 用户在图层操作菜单中点击 "停止编辑"。
   - 或在编辑状态下点击其他图层的 "开启编辑"。
   - 编辑状态取消，高亮和节点标识隐藏。
   - 如有未保存的更改，提示用户保存或放弃。

#### 5.1.3 UI 状态

| 状态       | 图层列表视觉            | 地图要素视觉      | 鼠标指针  |
| ---------- | ----------------------- | ----------------- | --------- |
| 非编辑状态 | 正常显示                | 正常样式          | default   |
| 编辑状态   | 蓝色高亮边框 + 铅笔图标 | 高亮 + 置顶渲染   | -         |
| 悬停点要素 | -                       | 点放大 + 边框高亮 | grab      |
| 悬停线段   | -                       | 线段加粗          | default   |
| 悬停节点   | -                       | 节点实心高亮      | move      |
| 可吸附位置 | -                       | 显示预览点        | crosshair |
| 拖动中     | -                       | 幽灵元素跟随      | grabbing  |

#### 5.1.4 图标使用

- 开启编辑：`Pencil` / `Edit`
- 停止编辑：`CheckCircle` / `Save`

### 5.2 缩放至图层

#### 5.2.1 功能说明

"缩放至图层"功能将地图视角调整到能够完整显示该图层所有要素的范围。

#### 5.2.2 交互流程

1. 用户在图层操作菜单中点击 "缩放至图层"。
2. 系统计算该图层的边界范围（Bounding Box）。
3. 地图视图平滑动画过渡到目标范围和缩放级别。
   - 2D 模式：调整中心点和 zoom 级别。
   - 3D 模式：调整相机位置和朝向。
4. 动画完成后，图层所有要素应在当前视口内可见。

#### 5.2.3 边界情况处理

| 情况             | 处理方式                                       |
| ---------------- | ---------------------------------------------- |
| 单个点要素       | 使用默认缩放级别（如 zoom=15） centered 在该点 |
| 空图层           | 无反应                                         |
| 跨国际日期变更线 | 特殊计算边界，避免视角错误                     |
| 3D 模式          | 调整相机高度和俯仰角，确保完整可见             |

#### 5.2.4 图标使用

- 缩放至图层：`ZoomIn` / `Focus` / `Maximize2`

### 5.3 属性表

#### 5.3.1 功能说明

"属性表"功能以表格形式展示图层中所有要素的属性数据，支持查看、编辑、筛选和批量操作。

#### 5.3.2 UI 布局

**位置**：地图底部抽屉（Drawer），从下向上滑出。

**组成结构**（从上到下）：

1. **标题栏**
   - 左侧：图层名称 + 要素总数（如 "成都市 - 贵阳 (128 个要素)"）。
   - 右侧：
     - "导出" 按钮（导出为 CSV/Excel）。
     - "关闭" 按钮（X）。

2. **工具栏**
   - 左侧：
     - 复选框（全选/取消全选）。
     - "删除选中" 按钮。
     - "导出选中" 按钮。
   - 右侧：
     - 搜索框（按属性值筛选）。
     - 字段选择器（显示/隐藏列）。
     - 排序按钮（升序/降序）。

3. **表格区域**
   - 行：每个要素一行。
   - 列：
     - 第一列：选择复选框。
     - 第二列：序号（只读）。
     - 后续列：属性字段（可编辑）。
     - 最后一列：操作（定位到要素、删除）。
   - 固定列：第一列（选择框）和最后一列（操作）固定显示。
   - 滚动：中间属性列支持横向滚动。
   - 虚拟滚动：大量数据时仅渲染可见行。

4. **分页器**（可选）
   - 每页显示条数选择（25/50/100/全部）。
   - 页码导航。
   - 总条数显示。

#### 5.3.3 交互流程

1. **打开属性表**
   - 用户点击 "属性表" 菜单项。
   - 底部抽屉滑出，加载并显示属性数据。

2. **查看属性**
   - 滚动浏览所有要素。
   - 横向滚动查看所有字段。

3. **编辑属性**
   - 点击单元格进入编辑模式。
   - 根据字段类型显示对应输入控件：
     - 文本：文本输入框。
     - 数字：数字输入框。
     - 日期：日期选择器。
     - 枚举：下拉选择框。
   - 失去焦点或按 Enter 保存修改。
   - 按 Esc 取消修改。

4. **筛选**
   - 在搜索框输入关键字，表格实时筛选。
   - 支持多字段联合搜索。

5. **排序**
   - 点击列标题切换升序/降序/无排序。

6. **定位到要素**
   - 点击行尾的 "定位" 按钮。
   - 地图视角缩放到该要素。
   - 要素高亮闪烁提示。

7. **删除要素**
   - 勾选要删除的要素。
   - 点击 "删除选中"。
   - 二次确认后执行删除。
   - 表格和地图同步更新。

8. **关闭属性表**
   - 点击 "关闭" 按钮或点击右上角 X。
   - 抽屉滑出隐藏。
   - 保持地图状态不变。

### 5.4 样式设置

#### 5.4.1 功能说明

"样式设置"功能提供图层可视化样式的配置界面，支持按几何类型（点/线/面）配置颜色、大小、透明度等属性。

#### 5.4.2 UI 布局

**位置**：地图右侧抽屉（Drawer），从右向左滑出。

**组成结构**（从上到下）：

1. **标题栏**
   - 左侧：图层名称 + 几何类型图标。
   - 右侧：
     - "重置样式" 按钮（恢复默认）。
     - "关闭" 按钮（X）。

2. **预设样式区**（可选卡片形式）
   - 展示 6-8 个预设样式卡片。
   - 点击卡片立即应用预设。
   - 当前应用的预设高亮显示。

3. **通用设置**（所有几何类型共有）
   - 最小可见缩放级别（Min Zoom）。
   - 最大可见缩放级别（Max Zoom）。
   - 不透明度（0-100% 滑块）。

4. **几何类型专属设置**

   **点图层**：
   - 颜色选择器（支持 solid 色和渐变色）。
   - 点大小（半径，单位：像素）。
   - 边框颜色。
   - 边框宽度。
   - 图标选择（可选内置图标或上传自定义图标）。

   **线图层**：
   - 颜色选择器。
   - 线宽（像素）。
   - 线型（实线/虚线/点线，下拉选择）。
   - 端点样式（圆形/方形，下拉选择）。
   - 拐角样式（圆角/尖角/斜角，下拉选择）。

   **面图层**：
   - 填充颜色选择器。
   - 填充不透明度（0-100% 滑块）。
   - 边框颜色选择器。
   - 边框宽度（像素）。
   - 边框线型（实线/虚线）。

5. **高级设置**（折叠面板）
   - 基于属性的样式规则（如按字段值分级设色）。
   - 标注设置（是否显示标注、标注字段、字体样式）。

6. **底部操作区**
   - "应用" 按钮：保存并关闭。
   - "取消" 按钮：放弃更改并关闭。
   - （实时预览模式下，更改即时生效，无需点击应用）

#### 5.4.3 交互流程

1. **打开样式面板**
   - 用户点击 "样式设置" 菜单项。
   - 右侧抽屉从屏幕外滑入。
   - 加载当前图层样式配置。

2. **选择预设**
   - 点击预设样式卡片。
   - 地图实时更新预览。
   - 当前预设卡片高亮。

3. **自定义样式**
   - 修改各项配置。
   - 地图实时预览效果。
   - 支持撤销（Ctrl+Z）。

4. **重置样式**
   - 点击 "重置样式"。
   - 二次确认。
   - 恢复系统默认样式。

5. **关闭面板**
   - 点击 "关闭" 或右上角 X。
   - 如有未保存更改，提示用户。

#### 5.4.4 图标使用

- 样式设置：`Palette` / `Settings`
- 颜色选择：`Circle` / `Droplet`
- 重置：`RotateCcw`
- 保存：`Save`
- 关闭：`X`
