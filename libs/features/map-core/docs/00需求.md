# 需求

- 能够应用在公众号制图流程中：
  - 项目空间管理：实现天地图的项目空间管理功能，包括创建、打开、保存、分享项目空间。
  - 数据管理：可以上传矢量数据（GeoJSON、Shapefile），并在地图上进行显示和编辑。

- 将烟草web端的功能迁移到mapai中
  - 图层管理：统一管理 GeoJSON、Imagery、3D Tiles；显隐、飞行、重命名
  - 时间轴（MVP）：播放/暂停、倍速、当前时间指示；实体 availability 支持
  - 画笔（MVP）：点/线/面/矩形/圆；撤销/清除；GeoJSON 导出；基础编辑
  - 卷帘（MVP）：影像左右卷帘；选择前后图层；滑条控制
  - 扩展：测量（距离/面积）、标注、书签、截图、场景快照

- 在实现多时相管理时，可以参考的平台：
  - 天地图
  - esri
