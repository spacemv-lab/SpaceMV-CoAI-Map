## 2. 组件规范

### 2.1 Button（按钮）

- **用途**：触发操作
- **变体**：
  - `default` - 主要操作（上传、保存）
  - `destructive` - 危险操作（删除）
  - `outline` - 次要操作（取消）
  - `ghost` - 行内操作（详情、复制）
- **状态**：default, hover, active, disabled
- **尺寸**：sm, md, lg

```tsx
<Button variant="default" size="md">上传数据</Button>
<Button variant="destructive" size="sm">删除</Button>
```

### 2.2 DataTable（数据表格）

- **用途**：展示数据集列表
- **列配置**：

| 列名 | 字段 | 宽度 | 对齐 |
|------|------|------|------|
| 序号 | index | 60px | center |
| 文件名 | name | auto | left |
| 几何类型 | type | 120px | center |
| 记录数 | recordCount | 100px | right |
| 文件大小 | fileSize | 100px | right |
| 数据来源 | source | 120px | center |
| 上传时间 | createdAt | 180px | center |
| 操作 | actions | 200px | center |

- **交互**：
  - 行 hover 高亮
  - 支持列排序
  - 支持分页

### 2.3 UploadModal（上传模态框）

- **用途**：文件上传入口
- **尺寸**：md (max-width: 500px)
- **组成**：
  - Dropzone 拖拽区域
  - 进度条组件
  - 格式提示文字

```tsx
<UploadModal
  accept=".geojson,.shp,.kml,.zip"
  maxSize={100 * 1024 * 1024}
  onUpload={handleUpload}
/>
```

### 2.4 StorageCard（存储卡片）

- **用途**：展示存储空间使用情况
- **布局**：
  - 环形图（左侧）
  - 数据详情（右侧）
- **数据**：
  - 总空间
  - 已使用
  - 使用率百分比

### 2.5 Progress（进度条）

- **用途**：上传/解析进度
- **状态**：
  - 进行中：蓝色动画
  - 完成：绿色
  - 失败：红色

## 3. 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| mobile | < 768px | 单列，全宽 |
| tablet | 768px - 1024px | 双列 |
| desktop | > 1024px | 左侧 75% + 右侧 25% |

## 5. 图标规范

使用 lucide-react 图标库：

```tsx
import { Upload, File, Trash2, Copy, Download, Eye } from 'lucide-react';

<Upload className="h-4 w-4" />  // 上传
<File className="h-4 w-4" />    // 文件
<Trash2 className="h-4 w-4" />  // 删除
<Copy className="h-4 w-4" />    // 复制
<Download className="h-4 w-4" /> // 下载
<Eye className="h-4 w-4" />     // 详情/预览
```
